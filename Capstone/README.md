# E-commerce Platform UI (React)

A responsive, professional e-commerce storefront interface built with React and Vite.

## Features
- Product catalog with category filters
- Instant search for efficient product discovery
- Cart management with quantity controls
- Responsive interface with polished visual styling

## Run Locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in the terminal.

## Project Files
- `src/App.jsx` — main UI and state management
- `src/components/Header.jsx` — search and category controls
- `src/components/ProductCard.jsx` — product cards
- `src/components/CartDrawer.jsx` — shopping cart panel
- `src/data/products.js` — sample product catalog
- `src/styles.css` — app styling
